# Clone do Netflix

Um clone do Netflix para estudos consumindo Api

## Tecnologias usadas

- React JS
- Api TMDB

## Para rodar aplicação
Crie variavél de ambiente 

crie arquivo .env

chaves

- REACT_APP_API_BASE = https://api.themoviedb.org/3
- REACT_APP_API_KEY = Aqui coloque sua chave da api tmdb
